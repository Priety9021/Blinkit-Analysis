# Blinkit-Analysis
**Recommended Structure and order**
# 1. Project title / Headline
Data-Driven Blinkit Sales Insights Dashboard<br>
Example:<br>
An interactive data visualization project built to analyze Blinkit grocery sales, focusing on category performance, revenue trends, customer behavior, and business insights
# 2. Short Description/ Purpose
This project analyzes Blinkit sales data to identify key trends, top-selling products, and category performance. The purpose is to transform raw data into interactive dashboards and support data-driven business decisions.

# 3. Tech Stack
List the key technologies used to build the dasboard<br>
Example:<br>
- Python – Pandas, NumPy (Data Cleaning & Analysis)
- Matplotlib / Seaborn – Data Visualization
- Excel – Data Preprocessing
- SQL
# 4. Data Source
Blinkit sales dataset in CSV format containing product details, categories, pricing, ratings, order quantities, and revenue information. The dataset was sourced from publicly available platforms (such as Kaggle) and used for learning and portfolio development.

# 5.Features / Highlights
- Cleaned and preprocessed raw Blinkit sales data using Python and Excel
- Performed Exploratory Data Analysis (EDA) to identify sales patterns and trends
- Built interactive Power BI dashboards with slicers for dynamic filtering
- Analyzed category-wise and product-wise sales performance
- Visualized monthly revenue trends using line, bar, and area charts
- Created KPI cards for total sales, average order value, and product count
- Identified top-selling products and underperforming categories
- Studied customer ratings and their impact on sales
- Delivered actionable business insights through data-driven visualizations

# 6. Visualizations Used

- Funnel Chart – Conversion and sales stage analysis
- Pie Chart – Category-wise sales distribution
- Line Chart – Monthly and yearly sales trends
- Column & Bar Charts – Product and category performance comparison
- Matrix Visual – Detailed product-level summary
- KPI Cards – Total Sales, Average Order Value, Product Count
- Slicers – Interactive filtering by category, region, and product

# 7.Business Impact & Insights

- Identified top-performing product categories, helping prioritize inventory planning and promotions.
- Discovered peak sales periods through trend analysis, supporting better demand forecasting.
- Highlighted underperforming products using funnel and KPI analysis, enabling optimization strategies.
- Revealed strong correlation between customer ratings and sales volume.
- Enabled region-wise performance comparison to improve targeted marketing efforts.
- Delivered real-time KPI visibility through interactive dashboards, supporting faster business decisions.Converted raw sales data into actionable insights using Power BI visualizations.
